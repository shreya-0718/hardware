### Week 1 - Falstad Circuit Simulator
Link to view: https://is.gd/Ezq7rZ

Video: https://github.com/user-attachments/assets/2b4a1d10-ee25-4e34-a407-9e8d289c1620

This is a simple RC circuit that has the three LEDs light up (at different times, decided on by their capacitor) and then fade when the power source is disconnected (with the switch). All the RC circuits I found while trying to research included transistors (which I don't understand fully, YET), so a lot of this project was just experimenting with different placements and eventually finding one that worked.
